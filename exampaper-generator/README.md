# 🧠 Examify AI Generator Service

This standalone repository houses the **Exam Generator Service** for the Examify Enterprise platform. It acts as an isolated microservice that integrates tightly with the core `examify-ai` backend to generate high-quality, authentic exam papers using a **Text-to-SQL** engine.

## 📖 Architecture & Design

Please refer to the [TECH_SPEC.md](TECH_SPEC.md) for detailed architectural diagrams, data flows, and expected input/output data models.

## 🛠️ Stack
- **Framework:** FastAPI (Python)
- **AI Core:** LangChain / LlamaIndex (Text-to-SQL)
- **Database:** PostgreSQL (Read-Only Integration)
